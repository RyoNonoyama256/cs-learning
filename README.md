# cs-learning

## Array (Dynamic Array)
→ 詳細: [array/README.md](array/README.md)

| 操作 | 計算量 | 理由 |
|------|--------|------|
| アクセス | O(1) | 先頭アドレス + index × sizeof で直接計算 |
| 末尾追加 | O(1)※ | ※ならし計算量。realloc時はO(n) |
| 挿入 | O(n) | 挿入位置以降の要素を全てずらす |
| 削除 | O(n) | 削除位置以降の要素を全て詰める |
| 検索 | O(n) | 先頭から順に探索 |

## Linked List (単方向)
→ 詳細: [linked-list/README.md](linked-list/README.md)

| 操作 | 計算量 | 理由 |
|------|--------|------|
| アクセス | O(n) | 先頭から順にたどる |
| 末尾追加 | O(1) | tailポインタで直接追加 |
| 先頭挿入 | O(1) | headを付け替えるだけ |
| index指定挿入 | O(n) | 挿入位置まで順にたどる |
| ノード指定挿入 | O(1) | ポインタの付け替えのみ |
| 削除 | O(n) | 単方向のためprevを探す必要がある |
| 検索 | O(n) | 先頭から順に探索 |
